## Gestion Utilisateurs

Cette application permet de gérer les utilisateurs : Ajouter, Modifier ou Supprimer un utilisateur

## Prérequis pour tester :

1. Importer la Base de *bdd.sql* se trouvant au niveau du répertorte courant

2. Définir les paramétres *USER* et *PASSWORD* au niveau de la classe *DatabaseConnection*  dans le répertoire *./src/main/java//utils/DataBaseConnection.java* représentant respectivement :   le login et le mot de passe d'utilisateur de votre SBGD.
